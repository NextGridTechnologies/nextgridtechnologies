How We Recovered a Law Firm From Ransomware in 2 Hours

Saturday 2:14am: Our EDR alert fires. Crypto.exe executing on file server.

Monday 9:00am: 12 lawyers are billing clients again. Zero ransom paid. Zero data loss.

This is exactly how we did it — and why 90% of SA firms wouldn't survive this attack.

The Attack Timeline

02:14 AM: EDR detects Crypto.exe on file server. Server auto-isolated from network.
02:15 AM: We remote in via backup admin account. Confirm encryption started on S: drive.
02:30 AM: Kill process, preserve memory dump for forensics. Client still asleep.
06:00 AM: Backup integrity check: Veeam cloud backup from 01:00 AM is clean.
08:00 AM: Restore to clean VM begins. 847GB, ~2 hours.
10:00 AM: Client notified. Systems testing with senior partner.
11:00 AM: All staff back online. Old infected server nuked from orbit.

Why They Survived: The 3-2-1 Backup Rule

We follow 3-2-1 religiously:

  • 3 copies: Live server + Onsite Synology NAS + Wasabi immutable cloud
  • 2 media types: Local disk + cloud object storage
  • 1 offsite: Air-gapped, 30-day retention, can't be deleted by ransomware

Backup vendor: Veeam for M365 + Synology Active Backup
Test frequency: Quarterly restore drills. Last test was 3 weeks before attack.

Why 90% of Firms Would Have Paid Ransom

  • No MFA on RDP: Intern used "Password123". Attacker brute-forced in.
  • Antivirus only: Old McAfee missed it. No EDR with rollback.
  • Backups never tested: 60% of firms have corrupt backups and don't know it.
  • No air-gap: Backups were on same network. Ransomware encrypted them too.

The Cost Breakdown

Our emergency response: R0 — included in R300/user/mo managed plan
Downtime cost avoided: ~R15,000 [2hrs × 12 lawyers × R625/hr]
If no backup: R150k+ ransom demand + 2 weeks rebuild + reputation damage

Checklist: Can Your Firm Survive This?

  • Immutable backups tested in last 90 days?
  • EDR with ransomware rollback, not just antivirus?
  • MFA enforced on all admin + RDP accounts?
  • Air-gapped copy offsite?
  • Incident response plan printed and accessible?

Score less than 4/5? You're one click away from disaster.
